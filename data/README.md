# Dataset Storage

This directory holds raw and processed ASL datasets. Follow the structure below.

## Expected Layout

```
data/
├── raw/                          # Downloaded datasets (can be deleted after preprocessing)
│   ├── how2sign/                 # How2Sign keypoints + annotations
│   ├── asllvd/                   # ASLLVD videos + token table
│   ├── asllrp_ncslgr/            # NCSLGR videos + ELAN annotations
│   └── wlasl/                    # WLASL clips + JSON metadata
│
└── processed/                    # Generated by preprocessing pipeline
    ├── keypoints/                # .npz skeleton files per dataset
    │   ├── how2sign/
    │   ├── asllvd/
    │   ├── asllrp/
    │   └── wlasl/
    └── manifests/                # .jsonl manifests + vocab
        ├── how2sign.jsonl
        ├── asllvd.jsonl
        ├── asllrp.jsonl
        ├── wlasl.jsonl
        ├── islr_train.jsonl
        ├── islr_val.jsonl
        ├── islr_test.jsonl
        ├── islr_goal_train.jsonl
        ├── islr_goal_val.jsonl
        ├── islr_goal_test.jsonl
        ├── cslr_train.jsonl
        ├── cslr_val.jsonl
        ├── cslr_test.jsonl
        ├── cslr_synthetic_train.jsonl
        ├── cslr_synthetic_val.jsonl
        ├── cslr_synthetic_test.jsonl
        ├── cslr_full_train.jsonl
        ├── cslr_full_val.jsonl
        ├── cslr_full_test.jsonl
        ├── islr_vocab.json
        ├── goal_vocab.json
        ├── cslr_vocab.json
        ├── cslr_synthetic_vocab.json
        ├── cslr_full_vocab.json
        ├── goal_manifest_metadata.json
        ├── cslr_synthetic_metadata.json
        └── label_maps/
            ├── wlasl_label_map.json
            ├── asllvd_label_map.json
            └── bu_corpus_label_map.json
```

## Download Sources

| Dataset | URL | Notes |
|---------|-----|-------|
| **How2Sign** | https://how2sign.github.io | Download 2D keypoints + annotations |
| **ASLLVD** | https://www.bu.edu/asllrp/av/dai-asllvd.html | Frontal view videos + token table |
| **NCSLGR** | https://www.bu.edu/asllrp/ncslgr-for-download/download-info.html | Videos + ELAN `.eaf` files |
| **WLASL** | https://github.com/dxli94/WLASL | Cropped clips + JSON metadata |

## Preprocessing

After placing raw data, run:

```bash
python scripts/preprocess.py --dataset <name> --config configs/preprocessing.json --clean-output
```

Where `<name>` is one of: `how2sign`, `asllvd`, `ncslgr`, `wlasl`.

Then rebuild the training manifests:

```bash
python scripts/build_training_manifests.py --config configs/preprocessing.json
python scripts/build_synthetic_cslr_manifests.py --manifest-dir data/processed/manifests
```

Notes:
- `wlasl.jsonl` and `how2sign.jsonl` are the canonical source manifests.
- `islr_goal_*.jsonl` is the focused top-10 live glossary subset and is stratified so every goal gloss appears in `val` and `test`.
- `cslr_synthetic_*.jsonl` is the recommended exact-label live CSLR path built from multi-word sequences synthesized out of goal-vocabulary WLASL clips.
- `cslr_*.jsonl` preserves the focused weakly supervised How2Sign path for comparison experiments.
- `cslr_full_*.jsonl` preserves the broader pseudo-labeled How2Sign path.
- In this workspace, How2Sign does not include gold gloss annotations, so CSLR
  labels are derived deterministically from the provided sentence text. The
  public How2Sign download script currently leaves gloss annotations as a TODO.
